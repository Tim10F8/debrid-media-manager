Debrid services
- Real-Debrid REST/OAuth: https://app.real-debrid.com (REST /rest/1.0, OAuth /oauth/v2/token) plus download hosts https://{id}.download.real-debrid.com/speedtest/test.rar/... for health checks; used in src/services/realDebrid.ts, src/pages/api/anticors.ts, src/pages/api/stremio/*, src/lib/observability/streamServersHealth.ts, src/lib/observability/streamServersList.ts, scripts/check_rd_servers.sh.
- Real-Debrid unrestrict proxy: https://unrestrict.debridmediamanager.com/ for link unrestrict/proxy flows (src/services/realDebrid.ts).
- AllDebrid API: https://api.alldebrid.com (v4/v4.1) for PIN login, user info, magnet upload/status/delete (src/services/allDebrid.ts, src/pages/api/anticors.ts).
- TorBox API: https://api.torbox.app (v1) for torrent upload and metadata (src/services/torbox.ts).

Metadata and discovery
- MDBList API: https://mdblist.com/api for search/info/list data (src/services/mdblistClient.ts and API routes that depend on it).
- Cinemeta/Stremio metadata: https://v3-cinemeta.strem.io/meta/* and /catalog/* (src/services/metadataCache.ts, api/info/movie.ts, api/info/show.ts).
- OMDb API: https://www.omdbapi.com search/detail endpoints (src/services/metadataCache.ts).
- TMDb API: https://api.themoviedb.org/3 for find/movie/tv/videos (src/services/metadataCache.ts, services/movieCleaner.ts, services/tvCleaner.ts, pages/api/info/*, pages/api/related/[mediaType].ts, pages/api/calendar.ts, pages/api/poster.ts).
- TMDb image CDN: https://image.tmdb.org/t/p/... for posters/backdrops rendered on the client (pages/movie/[imdbid]/info.tsx, pages/show/[imdbid]/info.tsx).
- Trakt API: https://api.trakt.tv for search/trending/popular/user/lists/calendar (src/services/metadataCache.ts, src/services/trakt.ts, pages/api/person/*.ts, pages/api/related/[mediaType].ts, pages/api/calendar.ts).
- Trakt OAuth: https://trakt.tv/oauth/authorize and https://api.trakt.tv/oauth/token (pages/api/trakt/auth.ts, pages/api/trakt/exchange.ts).
- Anime metadata: https://anime-kitsu.strem.fun/meta/series/... and https://anime-kitsu.strem.fun/catalog/anime/kitsu-anime-list/search=... (pages/api/info/anime.ts, pages/api/search/anime.ts). Commented alternative: https://api.jikan.moe/v4/anime (not currently called).
- Credentials: OMDb/TMDB keys via env vars `OMDB_KEY`/`TMDB_KEY` (.env/.env.local); MDBList key via `MDBLIST_KEY` (.env/.env.local); Trakt client id/secret via `TRAKT_CLIENT_ID`/`TRAKT_CLIENT_SECRET` (.env/.env.local) and exposed client id also set in next.config.js `publicRuntimeConfig.traktClientId`.

Hash list publishing and DMM-operated hosts
- GitHub REST API: api.github.com accessed via Octokit to write hash list HTML into debridmediamanager/hashlists (src/pages/api/hashlists.ts); resulting files served at https://hashlists.debridmediamanager.com/{uuid}.html.
- Poster/asset hosts: https://posters{0-9}.debridmediamanager.com (poster images) and https://static.debridmediamanager.com (Stremio addon assets) referenced across UI and addon endpoints.
- Credentials: GitHub PAT via env var `GH_PAT` stored in .env/.env.local for hash list publishing.

Other external calls
- Tor IP check: https://check.torproject.org/api/ip (renew-tor.sh).
- Placeholder/backfill media links requested at runtime: https://fakeimg.pl/..., https://source.unsplash.com/..., https://picsum.photos/..., https://www.youtube.com/embed/... and https://youtube.com/watch?... for poster/trailer fallbacks and embeds.

Debrid API credential locations
- Real-Debrid client id sourced from next.config.js `publicRuntimeConfig.realDebridClientId` (fallback in code); user OAuth tokens acquired at runtime and not persisted in the repo; optional env seed `REAL_DEBRID_CLIENT_ID` present in .env.
- AllDebrid uses user-supplied API keys at runtime; no keys stored in repo (.env only holds hostname defaults).
- TorBox uses user-supplied access tokens at runtime; host defaults come from next.config.js.

Torrent scrapers/indexers credential locations
- Jackett/Prowlarr base URLs and API keys pulled from env vars `JACKETT`, `JACKETT_KEY`, `PROWLARR`, `PROWLARR_KEY` in .env/.env.local (checked in src/pages/api/scrapers/*.ts before running scrape jobs).
